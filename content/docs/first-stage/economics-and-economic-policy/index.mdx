## この科目の位置づけ

経済学・経済政策は、価格、需要と供給、消費者行動、企業行動、国民所得、金融財政政策など、`グラフと因果関係` の理解が問われる科目です。診断士試験では、公式の丸暗記ではなく、図の動きや政策の効果を説明できるかが重要になります。

## 学習のポイント

- ミクロでは `弾力性`、`消費者行動`、`費用曲線`、`市場の失敗` を優先する
- マクロでは `国民所得計算`、`乗数`、`IS-LM`、`AD-AS`、`国際マクロ` を優先する
- 定義だけでなく、`どの曲線がどう動くか` を言葉で説明できることが重要
- 計算問題は少なく見えても、弾力性、均衡国民所得、貨幣市場などは得点差が出やすい

## この科目で問われること

この科目では、大きく次の 3 つが問われます。

1. 市場や国民経済の仕組みを正しく理解しているか
2. グラフや式を使って因果関係を追えるか
3. 政策や外部環境の変化が、どの変数をどう動かすか説明できるか

## ミクロ経済学で押さえること

ミクロ経済学では、`需要曲線` と `供給曲線` が全体の入口です。ここが弱いと、弾力性、余剰分析、市場均衡、関税の効果などが全部つながりません。

次に重要なのが、`消費者行動` と `企業行動` です。無差別曲線、予算制約線、限界代替率、費用曲線、限界費用の関係は、図で聞かれやすい典型論点です。

そのうえで、`市場構造と市場の失敗` までつなぐと、独占、外部性、公共財、情報の非対称性を一貫して理解しやすくなります。

### ミクロで特に優先したい論点

- 需要と供給のシフト要因
- 需要の価格弾力性、所得弾力性、交差弾力性
- 無差別曲線と予算制約線
- スルツキー分解
- 平均費用、限界費用、操業停止点
- 独占と完全競争の違い
- 外部性、公共財、余剰分析

## マクロ経済学で押さえること

マクロ経済学では、まず `国民所得計算` と `主要経済指標` を整理します。GDP、GNI、実質と名目、物価指数、失業率の違いが曖昧なままだと、後半の政策論点を正しく読めません。

その後、`財市場`、`貨幣市場`、`IS-LM`、`AD-AS` と進むと、財政政策と金融政策の効果を一連の流れとして理解できます。ここは診断士試験の得点源です。

さらに、`国際マクロ経済学` と `主要理論の比較` まで整理すると、為替や政策効果の違いを深く説明しやすくなります。

### マクロで特に優先したい論点

- GDP、GNI、三面等価
- CPI と GDP デフレーター
- 限界消費性向と乗数
- IS 曲線と LM 曲線のシフト要因
- 財政政策と金融政策の効果
- AD-AS 分析
- 国際収支、為替、マンデル=フレミング・モデル

## 典型的なつまずき

- 自財価格の変化による `曲線上の移動` と、外部要因による `曲線のシフト` を混同する
- ミクロの図を見ているのに、式だけで判断してしまう
- 名目と実質、CPI と GDP デフレーターの違いが曖昧なまま進む
- IS-LM を丸暗記して、どちらの曲線が何を表すか説明できない
- 政策効果を `短期` と `国際経済` の文脈で切り替えられない

## このページの次の拡張方針

- `ミクロ経済学` と `マクロ経済学` の単独ページを育て、図の読み方を補強する
- 図を読むときの観点を、論点ごとに短い箇条書きで追加する

## 学習メモ

- 文章だけで覚えず、図の形とシフト要因をセットで押さえる
- 1つの論点を `定義 → 図 → 典型問題` の順で整理する
- 国際マクロやゲーム理論のような後半論点も、早めに一覧で見えるようにしておく
