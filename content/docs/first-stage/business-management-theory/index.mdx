## この科目の位置づけ

企業経営理論は、`経営戦略論`、`組織論`、`マーケティング論` をまとめて扱う科目です。理論名やフレームワークを暗記するだけでは足りず、`何を分析する理論か`、`似た概念と何が違うか`、`どの場面で使うか` を見抜く力が求められます。

第1次試験では文章が長めで、理論の定義そのものより `事例文の状況に最も合う考え方を選べるか` が問われやすい科目です。第2次試験では、`事例Ⅰ` と `事例Ⅱ` の土台にもなります。

## 学習のポイント

- まず `経営戦略`、次に `組織`、最後に `マーケティング` の順で固める
- 理論名の丸暗記ではなく、`分析対象` と `使う場面` を覚える
- 似た理論は比較で覚える
- 第2次試験の `事例Ⅰ` と `事例Ⅱ` を意識して読む

## この科目で問われること

この科目では、主に次の 3 つが問われます。

1. 企業の状況に対して、どの理論やフレームワークを当てはめるべきか
2. 似た概念の違いを区別できるか
3. 戦略、組織、人事、顧客、販路の論点を、文脈に応じて説明できるか

## 章立て

### 経営戦略論

- 経営理念、ドメイン、SWOT 分析
- 多角化戦略、PPM、M&A、アライアンス
- 競争戦略、5フォース、3つの基本戦略、バリューチェーン
- RBV、VRIO、コアコンピタンス、ダイナミック・ケイパビリティ
- 技術経営、イノベーション、国際経営、CSR・ESG、DX

### 組織論

- 組織構造と部門化
- バーナード、サイモン、コンティンジェンシー理論
- モチベーション理論とリーダーシップ論
- 人的資源管理
- 組織文化、組織変革、ナレッジマネジメント

### マーケティング論

- マーケティングの基本概念
- STP と 4P
- 消費者行動とマーケティングリサーチ
- 製品、価格、流通チャネル、プロモーション
- サービスマーケティング、CRM、デジタルマーケティング

## 学習の優先順位

| 優先度 | 経営戦略論 | 組織論 | マーケティング論 |
| --- | --- | --- | --- |
| 最優先 | SWOT、PPM、5フォース、3つの基本戦略、バリューチェーン、VRIO | 組織構造、バーナード、サイモン、モチベーション理論、リーダーシップ論 | STP、4P、消費者行動、新製品開発、流通チャネル、プロモーション |
| 次に固める | 多角化、アンゾフ、M&A、アライアンス、イノベーション | 人的資源管理、組織文化、組織変革、コンフリクト | ブランド、価格戦略、サービスマーケティング、CRM |
| 後半で伸ばす | 国際経営、CSR・ESG、プラットフォーム、デジタル戦略 | 労働関連法規、心理的安全性、ダイバーシティ | デジタルマーケティング、共創、BtoB マーケティング |

## 典型的なつまずき

- `全社戦略` と `競争戦略` を混同する
- 5フォース、バリューチェーン、VRIO の `分析対象` を取り違える
- 組織構造の違いを、権限や調整の観点で説明できない
- モチベーション理論とリーダーシップ理論を一緒に覚えてしまう
- STP と 4P の前後関係が曖昧なまま、施策だけを覚えてしまう

## このページの次の拡張方針

- 章ページごとに、`頻出フレームワークの比較表` を追加する
- `事例Ⅰ` と `事例Ⅱ` に直結する論点には、横断導線を増やす
- 用語を並べるより、`どの設問で使うか` が見える説明を増やす

## 学習メモ

- 理論は `定義` と `使いどころ` をセットで覚える
- 似た理論は、目的、分析対象、典型的な設問で比較する
- 2次試験で使うときの言い換えまで意識しておくと崩れにくい
