## このページの役割

このページは、`マクロ経済学の中核` を固める解説ページです。`消費`、`投資`、`乗数` から `IS-LM`、さらに `財政政策` と `金融政策` の効果までを 1 本でつなげます。

## 学習のポイント

- まず `消費関数`、`投資`、`乗数` の意味を押さえる
- 次に `IS` は財市場、`LM` は貨幣市場と固定する
- そのうえで政策がどちらの曲線を動かすかを整理する

## 試験で何が問われるか

- 限界消費性向と乗数の関係を説明できるか
- `IS` と `LM` の意味を区別できるか
- 財政政策と金融政策が、所得と利子率にどう効くか読めるか
- クラウディングアウトの考え方を説明できるか

## 定義と仕組み

### 消費関数と限界消費性向

ケインズ型消費関数: `C = C₀ + cY`（`C₀`: 基礎消費、`c`: 限界消費性向、`Y`: 所得）

- `限界消費性向 c`: 所得が 1 増えたとき消費が増える量（`0 < c < 1`）
- `限界貯蓄性向 s = 1 - c`: 所得が 1 増えたとき貯蓄が増える量
- `平均消費性向 APC = C/Y = C₀/Y + c`: 所得が増えると `APC` は低下する

### 乗数の 3 パターン

乗数は、最初の支出増加が最終的に所得を何倍押し上げるかを表します。モデルの前提で式が変わります。

#### パターン 1: 定額税モデル（最も基本）

`Y = C₀ + c(Y - T) + I + G` を `Y` について解くと:

| 乗数 | 公式 | 意味 |
| --- | --- | --- |
| 政府支出乗数 | `1 / (1 - c)` | `G` が 1 増えたとき `Y` の増加 |
| 租税乗数 | `-c / (1 - c)` | `T` が 1 増えたとき `Y` の減少 |
| 均衡予算乗数 | `1` | `G` と `T` を同額増やすと `Y` も同額増加 |

例: `c = 0.8` なら政府支出乗数 = `1/(1-0.8) = 5`、租税乗数 = `-0.8/(1-0.8) = -4`

#### パターン 2: 比例税モデル

税額が所得に比例する場合（`T = tY`、`t`: 税率）:

| 乗数 | 公式 |
| --- | --- |
| 政府支出乗数 | `1 / (1 - c(1 - t))` |

例: `c = 0.8, t = 0.25` なら `1/(1 - 0.8×0.75) = 1/0.4 = 2.5`。定額税モデル（乗数 5）より小さくなります。これは、所得が増えるたびに比例税が自動的に増えて追加消費を抑えるためです（ビルトインスタビライザー）。

#### パターン 3: 開放経済モデル

輸入が所得に比例する場合（`M = mY`、`m`: 限界輸入性向）:

| 乗数 | 公式 |
| --- | --- |
| 政府支出乗数（比例税＋開放） | `1 / (1 - c(1 - t) + m)` |
| 政府支出乗数（定額税＋開放） | `1 / (1 - c + m)` |

例: `c = 0.8, t = 0.25, m = 0.1` なら `1/(0.4 + 0.1) = 2.0`。閉鎖経済（2.5）よりさらに小さくなります。

#### 乗数の大小関係

`定額税（閉鎖）> 比例税（閉鎖）> 比例税（開放）`

漏れ（税・貯蓄・輸入）が大きいほど乗数は小さくなります。

### 典型的な誤答パターン（乗数）

- 誤り: 均衡予算乗数は `c` に依存する → 正: 常に 1（`G` と `T` の乗数の和: `1/(1-c) + (-c/(1-c)) = 1`）
- 誤り: 開放経済では乗数が大きくなる → 正: 輸入という漏れが加わるので小さくなる
- 誤り: 比例税率 `t` を上げると乗数が大きくなる → 正: `t` が大きいほど `c(1-t)` が小さくなり、乗数は小さくなる

### 確認問題（乗数）

- Q1: `c = 0.75`、定額税モデルで政府支出を 100 増やしたとき、国民所得の増加は？
- A1: `100 × 1/(1-0.75) = 100 × 4 = 400`

- Q2: `c = 0.8, t = 0.2` の比例税モデルで政府支出乗数は？
- A2: `1/(1 - 0.8×0.8) = 1/(1 - 0.64) = 1/0.36 ≈ 2.78`

- Q3: 政府支出と税収をともに 50 増やした場合の所得変化は？（定額税モデル）
- A3: 均衡予算乗数 = 1 なので、所得も 50 増加

### デフレギャップとインフレギャップ

45 度線分析で、完全雇用 GDP（`Y_F`）と現実の均衡 GDP（`Y_E`）にずれがある場合:

| 概念 | 条件 | 意味 |
| --- | --- | --- |
| デフレギャップ | `Y_E < Y_F` | 総需要不足。非自発的失業が発生 |
| インフレギャップ | `Y_E > Y_F` | 総需要超過。物価上昇圧力 |

**デフレギャップの大きさ:** 完全雇用 GDP における `総需要の不足額`（= 45 度線上の `Y_F` の点と総需要線の垂直距離）

**ギャップの解消に必要な政府支出:**

`必要な政府支出増 = デフレギャップの大きさ`（乗数効果で `Y_F` まで回復する）

例: 完全雇用 GDP = 500、均衡 GDP = 400、乗数 = 5 の場合:
- GDP ギャップ = 500 - 400 = 100
- デフレギャップ = 100 / 5 = 20（必要な政府支出増は 20）

### リカードの等価定理

国債発行による財政支出は、将来の増税と等価であるため、消費に影響しないという定理です。

**論理:** 政府が減税して国債を発行 → 合理的な消費者は「将来の増税で返済される」と予想 → 減税分を貯蓄に回す → 消費は変わらない

**含意:** リカードの等価定理が成り立つなら、国債発行と課税は同等であり、財政政策の効果はない。

**成り立つ条件:**
- 消費者が完全に合理的で将来を見通せる
- 資本市場が完全（借入制約がない）
- 世代間の利他的連帯がある

**現実の限界:** 流動性制約のある家計、将来予測の不完全性、有限の寿命などにより、現実には完全には成り立ちにくい → 財政政策は一定の効果を持つ。

## 貨幣市場と信用創造

### 貨幣需要の 3 動機

ケインズは貨幣需要を 3 つの動機に分類しました。

| 動機 | 内容 | 依存する変数 |
| --- | --- | --- |
| 取引動機 | 日常の売買に必要な貨幣を持つ | 所得 `Y`（正の関係） |
| 予備的動機 | 予期しない支出に備えて貨幣を持つ | 所得 `Y`（正の関係） |
| 投機的動機 | 利子率変動による資産運用のために貨幣を持つ | 利子率 `r`（負の関係） |

貨幣需要関数: `L = L₁(Y) + L₂(r) = kY - hr`（`k`: 所得感応度、`h`: 利子率感応度）

利子率が下がると、債券価格が高くなり値下がりリスクが増すため、貨幣で持とうとする（投機的動機が増加）。利子率が極端に低いと全員が貨幣を選好し、貨幣需要が無限大 → `流動性のわな`。

### 貨幣数量説

古典派の貨幣理論で、貨幣量と物価の関係を説明します。

| 理論 | 公式 | 意味 |
| --- | --- | --- |
| フィッシャーの交換方程式 | `MV = PT` | 貨幣量 × 流通速度 = 物価 × 取引量 |
| ケンブリッジ方程式 | `M = kPY` | 貨幣需要は名目所得の一定割合 |

| 記号 | 意味 |
| --- | --- |
| `M` | 貨幣量 |
| `V` | 貨幣の流通速度（一定と仮定） |
| `P` | 物価水準 |
| `T`（or `Y`） | 取引量（or 実質所得） |
| `k` | マーシャルの `k`（= `1/V`） |

**古典派の結論:** `V`（or `k`）と `T` が一定なら、`M` の増加は `P` の同率上昇を招く → `貨幣の中立性`（貨幣量は実体経済に影響しない）。

**ケインズとの違い:** ケインズは投機的動機を導入し、利子率を通じて貨幣量が実体経済に影響しうるとした。

### 典型的な誤答パターン（貨幣需要）

- 誤り: 貨幣数量説では貨幣は実体経済に影響する → 正: 古典派の貨幣数量説では貨幣の中立性が成り立つ
- 誤り: 流動性のわなでは利子率が高い → 正: 利子率が下限に張り付いている状態
- 誤り: マーシャルの k が大きいほど貨幣の流通速度が速い → 正: `k = 1/V` なので逆の関係

### 貨幣乗数

中央銀行が供給する `ハイパワードマネー（H）` と市場の `マネーサプライ（M）` の関係を示します。

`M = m × H`（`m`: 貨幣乗数）

| 記号 | 意味 |
| --- | --- |
| `H` | ハイパワードマネー = 現金通貨 + 準備預金 |
| `c` | 現金・預金比率（= 現金 / 預金）※消費性向の `c` と別物 |
| `r` | 法定準備率（= 準備預金 / 預金） |
| `m` | 貨幣乗数 = `(1 + c) / (c + r)` |

例: `c = 0.1, r = 0.05` なら `m = 1.1/0.15 ≈ 7.33`

### 信用創造のメカニズム

1. 中央銀行が銀行 A に 100 万円を供給する
2. 銀行 A は準備率 `r` 分を残し、`100 × (1 - r)` を貸し出す
3. 貸出先が銀行 B に預金する → 銀行 B も準備率分を残して貸し出す
4. この連鎖で、最終的な預金総額 = `100 / r`

簡単な場合（現金保有なし）の信用創造額: `ΔD = ΔH / r - ΔH = ΔH × (1/r - 1) = ΔH × (1 - r) / r`

例: `ΔH = 100, r = 0.1` → 信用創造額 = `100 × 0.9/0.1 = 900`、預金総額 = `100/0.1 = 1000`

### 金融政策の手段

| 手段 | 操作 | H への影響 | M への影響 |
| --- | --- | --- | --- |
| 公開市場操作（買いオペ） | 国債を買う | `H` 増加 | `M` 増加 |
| 公開市場操作（売りオペ） | 国債を売る | `H` 減少 | `M` 減少 |
| 法定準備率引き下げ | `r` を下げる | 不変 | `m` 上昇で `M` 増加 |
| 政策金利引き下げ | 基準金利を下げる | 貸出増で `H` が動く | `M` 増加方向 |

### 典型的な誤答パターン（貨幣乗数）

- 誤り: 法定準備率を下げると貨幣乗数が下がる → 正: `r` が分母にあるので、`r` を下げると `m` は上がる
- 誤り: 信用創造額は最初の預金と同じ → 正: 準備率の逆数倍まで膨らむ
- 誤り: 貨幣乗数の `c` は限界消費性向と同じ → 正: 現金・預金比率であり全く別の概念

### 確認問題（貨幣乗数・信用創造）

- Q1: `r = 0.05`、現金保有なしの場合、100 万円のハイパワードマネーからの預金総額は？
- A1: `100 / 0.05 = 2000` 万円

- Q2: 現金・預金比率 `c = 0.2`、法定準備率 `r = 0.1` のとき貨幣乗数は？
- A2: `(1 + 0.2) / (0.2 + 0.1) = 1.2 / 0.3 = 4.0`

### IS 曲線

`IS 曲線` は財市場の均衡を表します。政府支出の増加や減税のように有効需要を押し上げる政策は、通常 `IS 曲線` を右へ動かしやすくなります。

#### IS 曲線の傾きを決める要因

IS 曲線は右下がりです。傾きの急緩は次の 2 つで決まります。

| 要因 | 大きいとき | IS 曲線 |
| --- | --- | --- |
| 投資の利子弾力性 `a`（利子率 1% 低下で投資がどれだけ増えるか） | 大きい | 緩やか（水平に近い） |
| 限界消費性向 `c` | 大きい | 緩やか（乗数が大きいため） |

IS が緩やかだと、金融政策（LM シフト）の所得増加効果が大きくなります。IS が急だと、財政政策（IS シフト）の利子率上昇が小さく、クラウディングアウトが小さくなります。

### LM 曲線

`LM 曲線` は貨幣市場の均衡を表します。貨幣供給増加のような金融緩和は、通常 `LM 曲線` を右へ動かしやすくなります。

#### LM 曲線の傾きを決める要因

LM 曲線は右上がりです。傾きの急緩は次の 2 つで決まります。

| 要因 | 大きいとき | LM 曲線 |
| --- | --- | --- |
| 貨幣需要の利子率感応度 `h` | 大きい | 緩やか（水平に近い） |
| 貨幣需要の所得感応度 `k` | 大きい | 急（垂直に近い） |

#### 極端なケースと政策効果

| ケース | LM の形 | 財政政策 | 金融政策 |
| --- | --- | --- | --- |
| 流動性のわな（`h → ∞`） | 水平 | 完全に有効（クラウディングアウトなし） | 無効（利子率が下限に張り付き） |
| 古典派ケース（`h = 0`） | 垂直 | 無効（完全クラウディングアウト） | 完全に有効 |

#### 典型的な誤答パターン（IS-LM の傾き）

- 誤り: IS が急だと財政政策が無効 → 正: IS が急だとクラウディングアウトが小さく、財政政策は比較的有効
- 誤り: 流動性のわなでは金融政策が有効 → 正: 無効。LM が水平なので LM を右にずらしても均衡が変わらない
- 誤り: LM が急なら財政政策が有効 → 正: LM が急（古典派ケース）だと完全クラウディングアウトで財政政策は無効

### 財政政策と金融政策

| 政策 | 主に動く曲線 | 典型的な読み方 |
| --- | --- | --- |
| 政府支出増加、減税 | `IS` | 所得を押し上げやすいが、利子率上昇も起こりやすい |
| 貨幣供給増加 | `LM` | 利子率を下げ、投資と所得を押し上げやすい |

財政政策では、利子率上昇によって民間投資が抑えられる `クラウディングアウト` に注意します。

## 典型例

- 限界消費性向が高いほど、乗数効果は大きくなりやすい
- 政府支出増加なら、通常は `IS` が右へ動く
- 金融緩和なら、通常は `LM` が右へ動く
- 財政政策だけで景気刺激をすると、利子率上昇が起こることがある

## 典型的なつまずき

- `IS` と `LM` の名前だけを覚え、何市場の均衡かを言えない
- 政策効果を `所得が増える` だけで終え、利子率の変化を落とす
- 乗数を計算式としてしか見ず、経済の連鎖として捉えない

## 問題を解くときの観点

- 財市場の話か、貨幣市場の話か
- 政策が最初に効くのは `需要`、`利子率`、`貨幣供給` のどれか
- 所得だけでなく利子率も問われていないか
- `短期の景気刺激` の話か `政策比較` の話か
