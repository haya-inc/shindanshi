## このページの役割

このページは、`マクロ経済学の後半論点` を固める解説ページです。`IS-LM` で学んだ政策効果を、`物価` と `国際経済` まで広げて読みます。

## 学習のポイント

- まず `AD` と `AS` の意味を区別する
- 次に `短期` と `長期` の違いを整理する
- そのうえで `フィリップス曲線` と `国際マクロ` を位置付ける

## 試験で何が問われるか

- `AD` と `AS` のどちらが動いたか判断できるか
- 物価と産出量を同時に読む問題に対応できるか
- フィリップス曲線の短期と長期の違いを説明できるか
- 閉鎖経済と開放経済で政策効果がどう変わるか比較できるか

## 定義と仕組み

### AD-AS 分析

`AD` は総需要、`AS` は総供給です。`AD` の変化は需要側の要因、`AS` の変化は供給側の要因として読むと整理しやすくなります。

`IS-LM` が所得と利子率を主に見るのに対し、`AD-AS` は物価と産出量を同時に見るところが違います。

### フィリップス曲線

短期では、インフレ率と失業率の間にトレードオフが見えやすいとされます。一方で長期では、その関係は安定しないと考えられます。ここは `短期と長期を混ぜない` ことが重要です。

#### 期待修正フィリップス曲線の標準式

`π = πe - β(u - u*)`

| 記号 | 意味 |
| --- | --- |
| `π` | 実際のインフレ率 |
| `πe` | 期待インフレ率 |
| `β` | 正の係数（失業率感応度） |
| `u` | 実際の失業率 |
| `u*` | 自然失業率（NAIRU） |

**短期フィリップス曲線:** `πe` を所与とすると、`u` が低いほど `π` が高い → 右下がり。金融・財政政策でトレードオフを利用できます。

**長期フィリップス曲線:** 長期では `πe = π`（期待が実現に一致）→ `u = u*` となり、失業率は自然失業率に固定 → 垂直。つまり長期的にはインフレ率と失業率のトレードオフは消滅します。

**自然失業率仮説（フリードマン）:** 長期的には失業率は自然失業率に収束し、金融政策でこれを恒久的に下げることはできない。

#### オークンの法則

実際の GDP と潜在 GDP の乖離（GDP ギャップ）と、失業率の乖離を結びつける経験則です。

`(Y - Y*) / Y* = -γ(u - u*)`

| 記号 | 意味 |
| --- | --- |
| `Y` | 実際の GDP |
| `Y*` | 潜在 GDP |
| `γ` | オークン係数（通常 2〜3） |
| `u - u*` | 循環的失業率 |

例: `γ = 2`、失業率が自然失業率を 1% 上回る → GDP は潜在 GDP を約 2% 下回る。

#### 典型的な誤答パターン（フィリップス曲線）

- 誤り: 長期フィリップス曲線は右下がり → 正: 垂直（自然失業率の位置）
- 誤り: 期待インフレ率が上がると短期フィリップス曲線が下にシフトする → 正: 上にシフトする（同じ失業率でもインフレ率が高くなる）
- 誤り: オークン係数が大きいほど失業の GDP への影響は小さい → 正: 大きいほど影響が大きい

#### 失業の種別

| 種別 | 原因 | 特徴 | 政策対応 |
| --- | --- | --- | --- |
| 摩擦的失業 | 転職の探索期間 | 自発的・短期的。景気に関わらず存在 | 職業紹介の効率化 |
| 構造的失業 | 産業構造の変化・技術変化 | スキルのミスマッチ。長期的 | 職業訓練・教育 |
| 循環的失業 | 景気後退による需要不足 | 非自発的。景気回復で解消 | 財政・金融政策 |

**自然失業率 = 摩擦的失業率 + 構造的失業率**（景気循環に依存しない部分）。循環的失業はゼロになりうるが、摩擦的・構造的失業はゼロにならない。

**完全雇用:** 循環的失業がゼロの状態（= 失業率が自然失業率に等しい状態）。「失業者がゼロ」ではない点に注意。

#### 確認問題（失業・フィリップス曲線）

- Q1: 自然失業率に含まれる失業の種類は？
- A1: 摩擦的失業と構造的失業。循環的失業は含まれない

- Q2: 期待インフレ率 3%、自然失業率 5%、β = 0.5 のとき、実際の失業率が 3% なら実際のインフレ率は？
- A2: `π = 3 - 0.5 × (3 - 5) = 3 + 1 = 4%`

### AS 曲線の導出と形状

AS 曲線の形は、賃金の硬直性に依存します。

| 立場 | 賃金の仮定 | AS の形状 | 政策含意 |
| --- | --- | --- | --- |
| 古典派 | 完全に伸縮的 | 垂直（長期 AS） | 需要政策は物価のみ変える |
| ケインズ派 | 下方硬直的 | 右上がり（短期 AS） | 需要政策で産出量も変わる |
| 極端なケインズ派 | 完全に固定 | 水平 | 需要政策が全て産出量に効く |

短期 AS が右上がりになる理由: 名目賃金が固定されているとき、物価上昇 → 実質賃金低下 → 企業の雇用増 → 産出量増加。長期では賃金が調整されるため AS は垂直に戻ります。

### インフレの分類と AD-AS での判別

| 種類 | 原因 | AD-AS での動き | 物価 | 産出量 |
| --- | --- | --- | --- | --- |
| デマンドプル・インフレ | 需要増加 | AD が右シフト | 上昇 | 増加 |
| コストプッシュ・インフレ | 供給コスト上昇 | AS が左シフト | 上昇 | 減少 |
| スタグフレーション | 供給ショック | AS が左シフト | 上昇 | 減少（同上） |

判別のコツ: 物価と産出量が `同方向` に動けばデマンドプル、`逆方向`（物価↑・産出量↓）ならコストプッシュ。

## 経済成長理論

### 成長会計（ソロー残差・TFP）

GDP 成長率を、労働・資本・技術進歩の寄与に分解する手法です。

**成長会計の公式:**

`ΔY/Y = ΔA/A + α × ΔK/K + (1 - α) × ΔL/L`

| 記号 | 意味 |
| --- | --- |
| `ΔY/Y` | GDP 成長率 |
| `ΔA/A` | 全要素生産性（TFP）成長率 = ソロー残差 |
| `α` | 資本分配率（通常 0.3〜0.4） |
| `ΔK/K` | 資本ストック成長率 |
| `ΔL/L` | 労働投入成長率 |

**計算手順:**
1. GDP 成長率、資本成長率、労働成長率の数値を読み取る
2. 資本分配率 `α` を確認する（問題で与えられる）
3. TFP 成長率 = `ΔY/Y - α × ΔK/K - (1 - α) × ΔL/L`

例: GDP 成長率 3%、資本成長率 4%、労働成長率 1%、`α = 0.3` → TFP = `3 - 0.3×4 - 0.7×1 = 3 - 1.2 - 0.7 = 1.1%`

### ソロー・モデルの基本

ソロー・モデルは、長期的な 1 人あたり所得を決める要因を分析するモデルです。

**前提:**
- 生産関数: `Y = A × K^α × L^(1-α)`（コブ＝ダグラス型）
- 1 人あたり生産関数: `y = A × k^α`（`y = Y/L, k = K/L`）
- 資本蓄積: `Δk = s × y - (n + δ) × k`（`s`: 貯蓄率、`n`: 人口成長率、`δ`: 減耗率）

**定常状態:**

`Δk = 0` → `s × A × k^α = (n + δ) × k` → `k* = (sA / (n + δ))^(1/(1-α))`

定常状態では 1 人あたり資本と 1 人あたり所得が一定になります。

**主要な結論:**

| パラメータ変化 | 定常状態の k* | 定常状態の y* | 長期成長率 |
| --- | --- | --- | --- |
| 貯蓄率 `s` 上昇 | 増加 | 増加 | 変わらない（水準効果のみ） |
| 人口成長率 `n` 上昇 | 減少 | 減少 | 変わらない |
| 技術進歩率 `g` 上昇 | — | — | 上昇（唯一の持続的成長源） |

**収束仮説:** 初期の資本が少ない国ほど成長率が高い（定常状態に向かって収束する）。

### ハロッド＝ドーマー・モデル

ソロー・モデル以前の成長モデルで、投資の二重性（需要効果と供給効果）を分析します。

**基本式:**

保証成長率: `Gw = s / v`（`s`: 貯蓄率、`v`: 資本係数 = `K/Y`）

| 概念 | 公式 | 意味 |
| --- | --- | --- |
| 保証成長率 `Gw` | `s / v` | 財市場が均衡を保つ成長率 |
| 自然成長率 `Gn` | `n + g` | 労働人口成長率 + 技術進歩率（供給の上限） |
| 現実の成長率 `G` | — | 実際の GDP 成長率 |

#### ナイフエッジの不安定性

ハロッド＝ドーマー・モデルの最大の特徴は、均衡成長経路が不安定であることです。

| 状況 | 起こること |
| --- | --- |
| `G > Gw` | 需要 > 供給 → 投資がさらに増加 → 均衡から一層乖離（好況加速） |
| `G < Gw` | 需要 < 供給 → 投資がさらに減少 → 均衡から一層乖離（不況深化） |
| `Gw > Gn` | 長期的に慢性的な不況（供給能力が需要を上回る） |
| `Gw < Gn` | 長期的にインフレ傾向（需要が供給能力を上回る） |

**ナイフの刃の上を歩くような不安定さ** → ソロー・モデルは資本係数 `v` を可変にして安定的な均衡を実現しました。

#### ソロー・モデルとの比較

| 項目 | ハロッド＝ドーマー | ソロー・モデル |
| --- | --- | --- |
| 資本係数 | 固定（`v` 一定） | 可変（資本と労働の代替可能） |
| 均衡の安定性 | 不安定（ナイフエッジ） | 安定（定常状態に収束） |
| 政策含意 | 成長率の偶然の一致が必要 | 技術進歩が持続的成長の鍵 |

### 典型的な誤答パターン（成長理論）

- 誤り: 貯蓄率を上げれば長期的に成長率が上がる → 正: 水準は上がるが定常状態での成長率は技術進歩率のみで決まる
- 誤り: TFP は直接観測できる → 正: 残差として計算される（「無知の尺度」とも呼ばれる）
- 誤り: ソロー・モデルでは政策で長期成長率を変えられる → 正: 変えられない。変えられるのは内生的成長理論

### 確認問題（成長理論）

- Q1: GDP 成長率 5%、労働成長率 2%、資本成長率 6%、資本分配率 0.4 のとき、TFP 成長率は？
- A1: `5 - 0.4×6 - 0.6×2 = 5 - 2.4 - 1.2 = 1.4%`

- Q2: ソロー・モデルで貯蓄率が上昇した場合、短期と長期で 1 人あたり所得の成長率はどうなるか？
- A2: 短期では一時的に上昇するが、長期では元の成長率（= 技術進歩率）に戻る

## 国際マクロ

### 国際収支の構成

| 勘定 | 内訳 | 具体例 |
| --- | --- | --- |
| 経常収支 | 貿易収支 | 財の輸出入 |
| | サービス収支 | 旅行、輸送、知財使用料 |
| | 第一次所得収支 | 海外投資の利子・配当（日本は大幅黒字） |
| | 第二次所得収支 | 無償援助、国際機関への拠出金 |
| 資本移動収支 | 直接投資、証券投資 | 海外 M&A、外国債購入 |
| 金融収支 | 外貨準備増減 | 中央銀行の為替介入 |

**基本等式:** 経常収支 + 資本移動収支 + 誤差脱漏 ≒ 0

**IS バランスとの関係:** 経常収支 = (S - I) + (T - G)（民間貯蓄投資差額 + 財政収支）

国際マクロでは、`為替相場`、`国際収支`、`資本移動` を前提に政策効果を考えます。

### 購買力平価説

為替レートの決定理論のうち、物価水準に着目するものです。

| 種類 | 公式 | 意味 |
| --- | --- | --- |
| 絶対的購買力平価 | `e = P / P*` | 為替レート = 自国物価 / 外国物価 |
| 相対的購買力平価 | `Δe/e ≈ π - π*` | 為替レートの変化率 ≈ インフレ率格差 |

`e`: 自国通貨建て為替レート（円/ドル）、`P`: 自国物価、`P*`: 外国物価、`π`: 自国インフレ率、`π*`: 外国インフレ率

**含意:** 自国のインフレ率が外国より高ければ、自国通貨は減価（円安）する。

**限界:** 非貿易財の存在、輸送費、関税などにより短期では成り立ちにくい。長期的な傾向の説明に使われます。

### 金利平価説（カバーなし金利平価）

`i - i* = (eᵉ - e) / e`

| 記号 | 意味 |
| --- | --- |
| `i` | 自国の名目金利 |
| `i*` | 外国の名目金利 |
| `eᵉ` | 予想将来為替レート |
| `e` | 現在の為替レート |

**含意:** 自国金利が外国より高い場合、将来の自国通貨減価が予想されている（高金利通貨は将来安くなる方向）。そうでなければ、皆が高金利国に投資して裁定機会が残るため。

### J カーブ効果

自国通貨安（円安）のあと、短期では経常収支が一旦悪化し、中長期で改善する現象です。

**メカニズム:**
1. 円安直後: 輸出数量・輸入数量はすぐに変わらない
2. 輸入の円建て価格が上昇 → 輸入金額が増加 → 経常収支悪化
3. 時間が経つと: 輸出数量増加・輸入数量減少 → 経常収支改善

時間軸でグラフを描くと J の字に見えることから名付けられました。

### マーシャル＝ラーナーの条件

通貨減価が経常収支を改善するための条件です。

`|εx| + |εm| > 1`

| 記号 | 意味 |
| --- | --- |
| `εx` | 輸出の価格弾力性 |
| `εm` | 輸入の価格弾力性 |

輸出と輸入の価格弾力性の絶対値の合計が 1 を超えるとき、通貨減価は経常収支を改善します。J カーブ効果の存在は、短期ではこの条件が満たされにくいことを示しています。

### 典型的な誤答パターン（為替理論）

- 誤り: 円安になれば経常収支はすぐに改善する → 正: 短期では J カーブ効果で悪化しうる
- 誤り: 高金利国の通貨は常に増価する → 正: 金利平価説では高金利国の通貨は将来減価が予想される
- 誤り: マーシャル＝ラーナー条件は弾力性の合計が 1 未満で改善 → 正: 1 超で改善

特に `マンデル=フレミング・モデル` は、開放経済での財政政策と金融政策の効き方を比較する論点です。

### マンデル＝フレミング・モデルの詳細

このモデルは IS-LM を開放経済に拡張し、為替制度による政策効果の違いを分析します。前提は `小国開放経済` と `完全資本移動`（国内利子率 = 世界利子率）です。

#### 変動為替制の場合

| 政策 | メカニズム | 最終効果 |
| --- | --- | --- |
| 財政拡大 | IS 右シフト → 利子率上昇圧力 → 資本流入 → 自国通貨高 → 輸出減・輸入増 → IS が左へ戻る | 所得は変わらない（無効） |
| 金融緩和 | LM 右シフト → 利子率低下 → 資本流出 → 自国通貨安 → 輸出増・輸入減 → IS も右シフト | 所得が増加（有効） |

#### 固定為替制の場合

| 政策 | メカニズム | 最終効果 |
| --- | --- | --- |
| 財政拡大 | IS 右シフト → 利子率上昇圧力 → 資本流入 → 為替維持のため外貨買い・自国通貨売り → LM も右シフト | 所得が増加（有効） |
| 金融緩和 | LM 右シフト → 利子率低下 → 資本流出 → 為替維持のため外貨売り・自国通貨買い → LM が左へ戻る | 所得は変わらない（無効） |

#### 覚え方

「変動→金融有効、固定→財政有効」。理由は、変動制では為替変動が政策効果を強化（金融）or 打ち消す（財政）のに対し、固定制では為替維持の介入が LM を動かして強化（財政）or 打ち消す（金融）ため。

### 典型的な誤答パターン（国際マクロ）

- 誤り: 変動為替制で財政拡大が有効 → 正: 通貨高による輸出減で相殺される
- 誤り: 固定為替制で金融緩和が有効 → 正: 為替維持の介入で LM が元に戻る
- 誤り: 経常収支 = 貿易収支 → 正: 貿易収支 + サービス収支 + 第一次所得収支 + 第二次所得収支

## 典型例

- 政府支出増加は、通常 `AD` を右へ動かしやすい
- 原材料価格高騰のような供給ショックは、`AS` を左へ動かしやすい
- 短期のフィリップス曲線では、失業率低下とインフレ率上昇が同時に見えることがある
- 開放経済では、為替や資本移動が政策効果を変える

## 典型的なつまずき

- `AD` と `AS` のどちらが動いたかを、結果だけ見て決めてしまう
- フィリップス曲線の短期と長期を混同する
- 国際マクロでも、閉鎖経済と同じ感覚で政策効果を読む
- 為替相場の変化と国際収支の方向を切り分けられない

## 問題を解くときの観点

- 需要ショックか、供給ショックか
- 物価と産出量のどちらが主に問われているか
- 短期の関係か、長期の関係か
- 閉鎖経済か、開放経済か
