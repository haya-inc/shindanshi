## このページの役割

このページは、`消費者行動` と `企業行動` をつなげて理解するための解説ページです。診断士試験では、無差別曲線の図と費用曲線の図が別々に出てきますが、どちらも `最適な選択` を問う図だと捉えると整理しやすくなります。

## 学習のポイント

- 消費者側では `無差別曲線`、`予算制約線`、`限界代替率` を固める
- 企業側では `生産関数`、`限界生産物`、`費用曲線`、`利潤最大化条件` を固める
- どちらも `どこが最適点か` を図で読めるようにする

## 試験で何が問われるか

- 最適消費点がどこに来るか判断できるか
- 所得変化と価格変化で予算制約線がどう動くか説明できるか
- 代替効果と所得効果を区別できるか
- `MC`、`AC`、`AVC` の位置関係と、損益分岐点、操業停止点を判断できるか
- 完全競争企業の利潤最大化条件を図で読めるか

## 定義と仕組み

### 無差別曲線と予算制約線

無差別曲線は、消費者にとって満足度が同じ財の組み合わせを表します。予算制約線は、所得と価格のもとで購入できる組み合わせの限界です。

最適消費点は、通常 `無差別曲線と予算制約線が接する点` に来ます。ここでの傾きの関係が `限界代替率 = 価格比` です。

### スルツキー分解

価格変化の効果は、`代替効果` と `所得効果` に分けて考えます。試験では、どの補助線を引くかよりも `相対価格が変わったから代替が起きる`、`実質所得が変わったから所得効果が出る` と説明できることが重要です。

#### スルツキー分解の 5 ステップ計算手順

1. **初期均衡を特定する**: 元の予算制約線と無差別曲線の接点 `E₀` を求める
2. **価格変化後の予算制約線を引く**: 変化した財の軸方向へ回転する
3. **補助予算線（スルツキー補助線）を引く**: 新しい価格比で、元の無差別曲線に接するように平行移動する
4. **代替効果を読む**: `E₀` → 補助線との接点 `E'` への動き。代替効果は常に価格下落した財の消費を増やす方向（符号が確定）
5. **所得効果を読む**: `E'` → 最終均衡 `E₁` への動き。正常財なら価格下落で実質所得増 → 消費増。ギッフェン財なら逆

#### スルツキー分解とヒックス分解の違い

| 項目 | スルツキー分解 | ヒックス分解 |
| --- | --- | --- |
| 補助線の基準 | 元の消費点を買える予算線 | 元の効用水準を維持する予算線 |
| 代替効果の特定 | やや大きめ | やや小さめ |
| 試験での出題 | 主流（補助線を引く問題） | 理論比較で出ることがある |

#### 財の分類と効果の向き

| 財の種類 | 代替効果 | 所得効果 | 全体効果 |
| --- | --- | --- | --- |
| 正常財（通常） | 価格下落 → 増加 | 実質所得増 → 増加 | 必ず増加 |
| 下級財（非ギッフェン） | 価格下落 → 増加 | 実質所得増 → 減少 | 代替効果 > 所得効果で増加 |
| ギッフェン財 | 価格下落 → 増加 | 実質所得増 → 大きく減少 | 所得効果 > 代替効果で減少 |

#### 典型的な誤答パターン（スルツキー分解）

- 誤り: 代替効果の向きは財の種類によって変わる → 正: 代替効果は常に価格変化と逆方向（法則）
- 誤り: ギッフェン財は下級財ではない → 正: ギッフェン財は下級財の特殊ケース（所得効果が非常に大きい下級財）
- 誤り: 補助線は元の無差別曲線に接する → 正: スルツキー分解では元の消費点を通り新価格比で引く（ヒックスは元の無差別曲線に接する）

### エンゲル曲線

エンゲル曲線は、所得と特定の財の消費量の関係を表す曲線です。横軸に所得、縦軸に消費量をとります。

| 財の種類 | エンゲル曲線の形状 | 所得弾力性 |
| --- | --- | --- |
| 正常財（必需財） | 右上がり（緩やか） | `0 < εI < 1` |
| 正常財（奢侈財） | 右上がり（急勾配） | `εI > 1` |
| 下級財 | 途中から右下がり | `εI < 0` |

**所得消費曲線との関係:** 所得変化で予算制約線が平行移動 → 最適消費点の軌跡 = 所得消費曲線。これを所得と消費量の平面に写したものがエンゲル曲線。

**エンゲルの法則:** 所得が増加すると食費の割合（エンゲル係数）は低下する → 食料品は必需財。

### 労働供給曲線と後方屈曲

労働供給にも代替効果と所得効果が働きます。賃金率の上昇に対して:

| 効果 | メカニズム | 労働供給への影響 |
| --- | --- | --- |
| 代替効果 | 余暇の機会費用が上昇 → 余暇を減らす | 労働供給増加 |
| 所得効果 | 実質所得上昇 → 余暇（正常財）の需要増 | 労働供給減少 |

**後方屈曲労働供給曲線:**
- 低賃金域: 代替効果 > 所得効果 → 賃金上昇で労働供給が増加（右上がり）
- 高賃金域: 所得効果 > 代替効果 → 賃金上昇で労働供給が減少（後方に屈曲）

後方屈曲が起こるのは、十分に高い所得水準に達すると、追加的な所得よりも余暇の価値が高くなるためです。

#### 典型的な誤答パターン（労働供給）

- 誤り: 賃金が上がれば労働供給は必ず増える → 正: 高賃金域では所得効果が代替効果を上回り減少しうる
- 誤り: 後方屈曲は非合理的な行動 → 正: 余暇を正常財として合理的に選好した結果

## 等量曲線と費用最小化

### 等量曲線（等生産量曲線）

等量曲線は、同じ生産量を達成できる `労働 L` と `資本 K` の組み合わせを表します。無差別曲線の生産者版です。

| 概念 | 消費者理論 | 生産者理論 |
| --- | --- | --- |
| 目的 | 効用最大化 | 費用最小化 or 生産量最大化 |
| 曲線 | 無差別曲線 | 等量曲線 |
| 制約線 | 予算制約線 | 等費用線 |
| 傾きの意味 | 限界代替率（MRS） | 技術的限界代替率（MRTS） |
| 最適条件 | MRS = 価格比 | MRTS = 要素価格比（`w/r`） |

### MRTS（技術的限界代替率）

`MRTS = MPL / MPK`（労働の限界生産物 / 資本の限界生産物）

費用最小化条件: `MRTS = w/r`（賃金率 / 資本のレンタル率）

### コブ＝ダグラス生産関数

`Q = A × L^α × K^β` の場合:
- `MPL = α × A × L^(α-1) × K^β = α × Q / L`
- `MPK = β × A × L^α × K^(β-1) = β × Q / K`
- `MRTS = (α/β) × (K/L)`
- 費用最小化: `(α/β) × (K/L) = w/r` → `K/L = (β/α) × (w/r)`
- `α + β > 1`: 規模に関して収穫逓増、`α + β = 1`: 収穫一定、`α + β < 1`: 収穫逓減

### 確認問題（等量曲線）

- Q1: `Q = L^0.5 × K^0.5`、`w = 4`、`r = 1` のとき、`Q = 10` を最小費用で生産する `L` と `K` は？
- A1: 費用最小化条件 `MRTS = w/r` → `(0.5/0.5) × (K/L) = 4/1` → `K = 4L`。`Q = L^0.5 × (4L)^0.5 = 2L = 10` → `L = 5, K = 20`。最小費用 = `4×5 + 1×20 = 40`

## 生産関数と費用曲線

### 限界生産物と平均生産物

- `MPL = ΔQ / ΔL`（労働の限界生産物）: 追加 1 単位の投入で増える生産量
- `APL = Q / L`（労働の平均生産物）: 1 単位あたりの生産量
- `MPL > APL` のとき `APL` は上昇、`MPL < APL` のとき `APL` は下降
- `MPL = APL` のとき `APL` は最大（限界が平均の最高点を通る）

### 収穫逓減の法則

他の投入を固定して 1 つの投入だけを増やすと、ある点から限界生産物が減少します。これが費用曲線の形状を決めます。

### MC 曲線の形状と根拠

MC（限界費用）曲線が U 字型になる理由は、生産関数から導けます。

**MC と限界生産物の関係:**

`MC = w / MPL`（`w`: 賃金率、`MPL`: 労働の限界生産物）

労働投入の初期段階では `MPL` が上昇するため `MC` は下がり、収穫逓減が始まると `MPL` が低下して `MC` が上がります。これが U 字型の根拠です。

### MC・AC・AVC の位置関係

| 関係 | 意味 |
| --- | --- |
| `MC < AVC` | AVC は下降中 |
| `MC = AVC` | AVC の最低点（= 操業停止点） |
| `AVC < MC < AC` | AVC は上昇中、AC はまだ下降中 |
| `MC = AC` | AC の最低点（= 損益分岐点） |
| `MC > AC` | AC は上昇中 |

**なぜ MC は AC・AVC の最低点を通るか:** 平均値が下がっているときは限界値 < 平均値、上がっているときは限界値 > 平均値。したがって限界値 = 平均値のとき平均値が最低になります（テストの平均点と同じ論理）。

### 損益分岐点と操業停止点

| 概念 | 条件 | 判断 |
| --- | --- | --- |
| 損益分岐点 | `P = AC の最低点` | この価格以上なら利潤 ≥ 0 |
| 操業停止点 | `P = AVC の最低点` | この価格未満なら生産停止（固定費すら回収不可） |

**短期供給曲線:** 完全競争企業の短期供給曲線は、`AVC の最低点以上の MC 曲線` です。`P < AVC` の領域では供給量 = 0。

### 費用曲線の計算例

総費用関数 `TC = Q³ - 6Q² + 15Q + 10` のとき:

1. `FC = 10`（Q = 0 のときの TC）
2. `VC = Q³ - 6Q² + 15Q`
3. `MC = dTC/dQ = 3Q² - 12Q + 15`
4. `AVC = VC/Q = Q² - 6Q + 15`
5. `AVC` の最低点: `dAVC/dQ = 2Q - 6 = 0` → `Q = 3`、`AVC = 9 - 18 + 15 = 6`
6. 操業停止価格 = 6

### 典型的な誤答パターン（費用曲線）

- 誤り: MC が AC の最低点を通るのは偶然 → 正: 数学的に必然（限界値と平均値の関係）
- 誤り: 操業停止点は AC の最低点 → 正: AVC の最低点（固定費は埋没費用なので判断に含めない）
- 誤り: 供給曲線は MC 曲線全体 → 正: AVC の最低点以上の部分のみ

### 確認問題（費用曲線）

- Q1: `TC = Q² + 4Q + 16` のとき、損益分岐点の価格は？
- A1: `AC = Q + 4 + 16/Q`、`dAC/dQ = 1 - 16/Q² = 0` → `Q = 4`、`AC = 4 + 4 + 4 = 12`。損益分岐価格 = 12

- Q2: `MC = w / MPL` で、`w = 1000`、`MPL = 50` のとき MC は？
- A2: `1000 / 50 = 20`

### 長期平均費用曲線（LAC）

短期では固定費が存在しますが、長期ではすべての費用が可変です。各生産規模での最小平均費用を包絡したものが `LAC` です。

| LAC の形状 | 意味 | 原因 |
| --- | --- | --- |
| 右下がり部分 | 規模の経済 | 分業、大量購入、技術的効率 |
| 最低点 | 最適規模 | 規模の経済と規模の不経済が釣り合う |
| 右上がり部分 | 規模の不経済 | 管理の複雑化、コミュニケーションコスト |

### 生産関数と費用曲線

企業行動では、投入量と産出量の関係が `生産関数`、費用との関係が `費用曲線` です。短期では固定費が存在するため、`平均固定費` は右下がりになりやすく、`限界費用` は平均費用や平均可変費用の最低点を通ります。

### 利潤最大化

完全競争企業では、価格は所与です。そのため、利潤最大化条件は `価格 = 限界収入 = 限界費用` で整理できます。

| 論点 | 見るべき点 | 典型問題 |
| --- | --- | --- |
| 無差別曲線 | 右下がり、原点に凸 | 最適消費点 |
| 予算制約線 | 傾きは価格比 | 所得変化、価格変化 |
| スルツキー分解 | 代替効果と所得効果 | 補助線の意味 |
| 費用曲線 | `MC` が `AC` と `AVC` の最低点を通る | 損益分岐点、操業停止点 |
| 利潤最大化 | `MR = MC` | 供給量の決定 |

## 典型例

- 所得増加なら予算制約線は平行に外側へ動く
- ある財の価格下落なら、その財の軸方向へ予算制約線が回転する
- `MC` が `AVC` の最低点を通るなら、その価格が操業停止点になる
- 完全競争企業では、価格が限界費用を上回る数量まで生産する

## 典型的なつまずき

- 無差別曲線の図を `需要曲線` と同じ感覚で読んでしまう
- 代替効果と所得効果の向きを、価格変化の方向と混同する
- `AC` と `AVC` を区別せず、操業停止点と損益分岐点を取り違える
- `MR = MC` を覚えていても、完全競争と独占で `MR` の形が違うことを見落とす

## 問題を解くときの観点

- 消費者の最適化か、企業の最適化か
- 変化したのは `所得`、`価格`、`投入量` のどれか
- 図で見ているのは `接点` か `交点` か
- 企業問題では、固定費が関わる短期なのか、より一般的な利潤最大化なのか
